# Hard Version

## Key Points

* Robot Shuffle
* Box Step Variation
* Pausing (and optionally isolations and grooving)

## Steps

1. Running Man x8 - starting on right foot
2. Robot Shuffle x4 -
3. Box Step Variation x2
4. Pause x2 counts
5. Groove x2 counts
6. Running Man x4

# Easy Version

## Demo

![Easy Demo](/posts/not-tonight-lil-kim/easy.mp4)

## Key Points

* Running Man
* T-Step
* Box Step
* Pausing (and optionally isolations and grooving)

## Steps

1. Running Man x8 - starting on right foot
2. T-Step x4 - to the right
3. T-step x4 - to the left
4. Box Step x2
5. Pause x2 counts
6. Groove x2 counts
7. Running Man x4
